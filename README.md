Financial fraud poses a major threat to banking institutions, e‑commerce platforms, and digital payment systems. This project presents a comparative study of three widely used machine learning algorithms—Logistic Regression, Random Forest, and Gradient Boosting—to evaluate their effectiveness in detecting fraudulent financial transactions.

The goal is to analyze model performance, interpretability, computational cost, and suitability for real‑world fraud detection systems.

📌 Objectives
Build fraud detection models using three ML algorithms

Compare performance using standard evaluation metrics

Analyze trade‑offs between accuracy, interpretability, and computational efficiency

Provide insights for selecting the best model for production‑level fraud detection

📊 Algorithms Compared
1. Logistic Regression
Linear model

Fast, interpretable, and easy to deploy

Performs well with linearly separable data

Often used as a baseline in fraud detection

2. Random Forest
Ensemble of decision trees

Handles non‑linear relationships

Robust to noise and overfitting

Provides feature importance
